What's Going on Here?
The minimum value in a bar chart (also called the baseline) should always be 0. When this value is greater than 0, the graph is said to have a truncated y-axis. As you can see in the example above, by truncating or extending the y-axis (making the range of y-values smaller or larger), it's possible to make bars appear to have almost any relationship to each other. Thing 2 is always bigger than Thing 1, but the relationship between them appears to change dramatically if the baseline is non zero. The actual values aren't changing! Graphs with non-zero baselines mislead the reader (whether or not that's the autors intent), representing relationships that don't exist.
